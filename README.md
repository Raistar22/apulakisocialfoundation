# Apulaki Social Foundation

Apulaki Social Foundation is a non-profit organization dedicated to **Trees & Biodiversity Conservation**. This web application serves as our digital platform to showcase our activities, share research findings, and facilitate community engagement and donations.

## 🌿 Our Mission

Our commitment to biodiversity conservation is backed by extensive fieldwork and research. We focus on:
- **Tree Plantation Drives**: Reforestation and urban greening initiatives across Maharashtra.
- **Biodiversity Research**: Documenting and protecting local flora and fauna.
- **Community Engagement**: Awareness programs and community-led conservation efforts.

## 🚀 Key Features

- **Activity Gallery**: A comprehensive view of our tree plantation drives and community events.
- **Research & Publications**: Access to our latest findings, including the "Butterflies of Waghapur" study.
- **Seamless Donations**: Support our cause through secure UPI or Bank Transfer methods.
- **Responsive Design**: A premium, modern web experience accessible on all devices.

## 🛠️ Technology Stack

- **React**: Modern UI development.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first styling for a sleek design.
- **shadcn/ui**: High-quality, accessible UI components.
- **Lucide Icons**: Clean and consistent iconography.

## 💻 Local Development

To run the project locally, follow these steps:

1. **Clone the Repository**:
   ```sh
   git clone https://github.com/Raistar22/apulakisocialfoundation.git
   cd apulakisocialfoundation
   ```

2. **Install Dependencies**:
   ```sh
   npm install
   ```

3. **Start Development Server**:
   ```sh
   npm run dev
   ```

The application will be available at `http://localhost:8080`.
